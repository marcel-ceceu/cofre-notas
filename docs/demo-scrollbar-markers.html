Marcadores de busca na rolagem — 3 sugestões

Mesmo conteúdo e mesmos highlights amarelos. Compare o visual de cada abordagem. Clique nos ticks/faixas para saltar até o trecho.

Opção 1 — Overview ruler (ticks ao lado)

Barra nativa intacta + traços amarelos na borda direita, estilo Firefox/VS Code.

Recomendada p/ o Cofre

Os ticks ficam sobre/ao lado da scrollbar. Clique num tick amarelo para ir ao match.

Opção 2 — Scrollbar customizada com markers

Track próprio; thumb + ticks amarelos no mesmo canal.

Mais integrado visualmente

Arraste o thumb ou clique no track. Os ticks amarelos vivem dentro da barra.

Opção 3 — Minimap do documento

Prévia estreita com faixas amarelas + retângulo do viewport (estilo editor).

Mais rico · mais espaço

Clique no minimap para saltar. O retângulo roxo mostra a região visível.
